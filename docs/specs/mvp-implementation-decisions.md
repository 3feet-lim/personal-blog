# MVP 구현 결정사항

## 범위 확정

- 공개 기술 블로그는 로그인 없이 읽을 수 있습니다.
- 가족 앨범은 전역 가족 접근 경계로 보호합니다.
- MVP에서 콘텐츠 관리와 자산 업로드는 관리자만 수행할 수 있습니다.

## 접근 정책

- `anonymous`
  - 랜딩 페이지와 블로그를 볼 수 있습니다.
  - 앨범과 관리자 라우트에는 접근할 수 없습니다.
- `family`
  - 가족 앨범을 볼 수 있습니다.
  - 자산 업로드나 콘텐츠 관리는 할 수 없습니다.
- `admin`
  - 가족 앨범을 볼 수 있습니다.
  - 블로그, 앨범, 사용자, 자산을 관리할 수 있습니다.

## 인증 결정

- OAuth 제공자 모델은 특정 제공자에 종속되지 않아야 합니다.
- 첫 번째 제공자는 Google입니다.
- 두 번째 후보 제공자는 Naver입니다.
- 자체 가입은 기본적으로 열어두지 않습니다.
- 가족 영역 접근은 사전 승인된 사용자에게만 부여합니다.

## 콘텐츠 결정

- 블로그 작성 포맷은 Markdown입니다.
- 가족 앨범 MVP는 이미지 전용입니다.
- 동영상 지원은 1차 구현 범위에서 제외합니다.

## 보호 자산 정책

- 가족 앨범 자산은 기본적으로 비공개입니다.
- 자산 URL은 공개 정적 경로로 노출하지 않고, 반드시 백엔드를 통해 발급해야 합니다.
- 저장소 코드는 S3 계열 백엔드와 호환 가능한 형태를 유지해야 합니다.
